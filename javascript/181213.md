### 생활코딩을 보면서 javascript를 연습합니다.

- 실행방법과 실습환경 / 숫자와 문자

```javascript
<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8"/>
    </head>
    <body>
        <script>
            alert('Hello world');
            alert("coding everybody"); //String은 '', "" 둘 다 가능
            alert('coding everybody');
            alert(typeof "1"); //typeof는 데이터의 형을 알려주는 기능
            alert(typeof 1);
            alert('misun\'s javascript') //문자열 내에 ' 를 쓰기
            alert("안녕하세요. \n 생활코딩의 세계에 오신것을 환영합니다.") //줄바꿈
            alert("coding"+"everybody");
            alert("coding everybody".length);

        </script>
    </body>
</html>

구글 크롬 브라우저에서는 command + alt + J를 누르면 웹 페이지에서 발생한 오류를 보여준다.
=> Syntax Error!
```
